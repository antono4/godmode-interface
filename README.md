# 🐣 GodMode Interface

A web-based interface for accessing multiple AI chat services simultaneously, inspired by [smol-ai/GodMode](https://github.com/smol-ai/GodMode).

## Features

- **Multi-AI Interface**: View ChatGPT, Claude, Perplexity, Gemini, and more in a split-pane layout
- **Synchronized Input**: Type once, explore across all AI services
- **Dark Mode UI**: Modern, clean interface with dark theme
- **Responsive Layout**: Flexible split-pane design
- **Keyboard Shortcuts**: Fast navigation with keyboard shortcuts

## Supported AI Services

- 🤖 ChatGPT (chat.openai.com)
- 🧠 Claude (claude.ai)
- 🔍 Perplexity (perplexity.ai)
- ✨ Gemini (gemini.google.com)
- 💻 Microsoft Copilot (copilot.microsoft.com)

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| `Enter` | Send message |
| `Shift + Enter` | New line |
| `⌘ R` | Start new conversation |
| `⌘ S` | Toggle provider selector |

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
